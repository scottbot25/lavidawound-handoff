Wound Handoff Builder — prototype

Parse one finished PCC note, verify the fields, add the patient to today's running list, then generate a clean handoff sheet for the facility.

Handles PHI. This prototype keeps everything in memory only — nothing is stored or sent, and refreshing clears it. For real use it must run inside your M365/HIPAA environment, with exports saved to the SharePoint vault and delivered to the facility over a secure channel.
Internal care-coordination tool — not an official record. The handoff sheet this produces is a working summary to help facility nursing staff coordinate wound care. It is not an official medical record, provider note, or billing document. The signed note in the EHR remains the authoritative record.
